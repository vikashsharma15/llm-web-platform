# Interactive Story Generator (Frontend)

A React-based interactive story game where users generate AI-powered branching stories and play through different decision paths.

Built with:
- React
- Vite
- Axios
- React Router

---

## Features

- 🎨 Generate story based on a theme
- ⏳ Background job polling system
- 🌳 Branching story structure (decision-based gameplay)
- 🏆 Multiple endings (Winning / Normal)
- 🔁 Restart story
- 🆕 Generate new story
- ⚡ Fast Vite dev environment
